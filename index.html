Mod wallz

Apache mod to mitigate Layer 7 DDoS attacks

Download as .zip Download as .tar.gz View on GitHub

mod_wallz home page.

mod_wallz is a simple project that aims to stop layer 7 DDoS attacks such as GET/POST abuse, slow post (slowloris) attack, and script injections.

How to install:

$ git clone https://github.com/LolDatSec/mod_wallz
$ ./install.sh

At current state, it doesn't protect from any attacks, don't install.
